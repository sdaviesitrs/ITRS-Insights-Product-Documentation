Install and Set-up

To utilize , you need an installation of both the platform (a single node or a cluster) and also the application.

Install Valo

If your administrator has already installed , you can jump to Install [%=General.Product Name Insights (abbreviated)%].

Requirements

Operating System:

  • Ubuntu Server 14.04 LTS
  • Red Hat Enterprise Linux 7
  • SUSE Linux Enterprise Server 12

Java:

An Oracle JRE 8 must be installed prior to running Valo. We test against JRE 8u51 but newer versions should also work fine.

Warning:
Versions 8u40 and 8u45 have serious problems when running Valo.

Install

Running a single node cluster is easy since the settings are pre-configured in the download packages. There are DEB and RPM packages pre-built. A ZIP is also provided for Windows and to run manually on Linux systems.

Start Valo

Simply extract the zip and run: ./bin/valo-cluster-app

Install

Requirements

  • Windows 7 and up or
  • Ubuntu 14.04 or
  • OS X 10.10 and up

Unzip and Run the Insightsv1.0.0.exe

Configuration

First you need to point your Insights at the correct Tenant of the instance you will use.

  • If you installed on your local machine then the default localhost:8888 will work in the http+sse: box.
  • If your admin installed for you, then post the URI that they will have given you in that box instead.

If you already have a Tenant set up on your instance of that you wish to access, then you need to enter that here. Alternatively, you can create your own Tenant from the log-in screen. See How to Create a Tenant.

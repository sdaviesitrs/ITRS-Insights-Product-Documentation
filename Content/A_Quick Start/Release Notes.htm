Improvements

User Interface

The colour scheme for has been improved. We've added more colours to the palette for Notebooks and query visualisations. Chart parameters have been consolidated into one section and a unified style applied.

General UI

Git Issue #1460Data in a Table could sometimes be presented in inverse order when sorting by ascending or descending value. This has been corrected.

Git Issue #650Scroll bars for visualisations in Dashboards have been revised to deliver a better user experience.

Performance

visualisation

Performance has been dramatically improved when processing and visualising large sources of data. Graphs that previously caused performance to degrade are now displayed without a problem. Some features have been removed while we re-engineer components for higher performance.

Known Behaviours

Performance

Nested data in tables

The previous table plug-in had performance issues displaying nested data. This functionality has been removed while we work on an alternative solution. Nested data will be displayed as a preview in order to process flattened data.

The workaround is for queries to be adjusted as follows:

This query will retrieve nested data:

from /streams/demo/geneos/cpu

This query uses the select clause and will retrieve flattened data:

from /streams/demo/geneos/cpu
select host.name host.id data.row.cpuutilisation

A solution to display nested data without incurring performance issues is planned.

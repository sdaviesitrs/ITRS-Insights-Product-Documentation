Dashboard layout customizing

IKABO-947

The layout for Dashboards is now fully customizable.

Simply click and drag a visualization panel header to move it, or click and drag the edge to resize it.

Many more features are planned for dashboards, but in the meantime 's flexible layout will allow dramatically more options when creating and customizing your Dashboards.

See: Organize a Dashboard

Bubble Map Chart

IKABO-961

We've added a Bubble Map chart that you can use with any query returning Geopoint data. For example:

from /streams/demo/equities/exchanges
select name, latency, geopoint(lat, lon)

Bubble maps should be used when showing density of information or events with a geographical context.

See: Bubble Map

KPI Chart

IKABO-858

A new KPI chart lets you rapidly create visualizations of your key metrics for subsequent placement on a Dashboard.

View the data as a number, column or sparkline. Alternatively, combine the number setting with the status setting to display a color-coded arrow indicator that allows super-fast understanding of whether the KPI trend is good or bad.

See: KPI

Bivar

Bivariate correlative statistics for use with 2 variables.

Format

bivar(x:Double,  y:Double)

Outputs

Name Type Meaning
n Int  
meanX Double mean value of x
meanY Double mean value of y
determinant Double  
varX Double variance of x
varY Double variance of y
covarXY Double covariance of x and y
slopeYX Double  
interceptYX Double  
slopeXY Double  
interceptXY Double  
pearsonsR Double correlation coefficient
covariance = a measure of how much 2 random variables x & y change together
pearsonsR = the correlation coefficient = a measure of the strength and direction of the linear relationship between two variables that is defined as the (sample) covariance of the variables divided by the product of their (sample) standard deviations.”

Examples

For example if we have a stream of data on USA flights we can see how correlated the distance travelled is to the time taken; as we might want to go with the airline most likely to get us there in the quickest time! The following takes the data where the destination is "LAX", putting into the Bivar function the Distance and Elapsed Time (both converted to doubles with functions documented in the Query Functions section) and using the As command to create an alias of correl for the output. It is then easy to filter/select flights based on the correlation coefficient by comparing the pearsonsR output fields (correl.pearsonsR). In this example we have said where >0.001 as > 0 can sometimes causes issues and so should be used with care.

from historical /streams/demo/usa/flightdata
where Dest == "LAX"
group by flightDate, UniqueCarrier
select flightDate, UniqueCarrier, bivar( toDouble(Distance),  toDouble(seconds(ActualElapsedTime))) as correl
select flightDate, UniqueCarrier, correl.pearsonsR as PR
where PR > 0.001
order by flightDate

The output of this can then be viewed in a number of the different graphs to show the comparisons of the various carriers.

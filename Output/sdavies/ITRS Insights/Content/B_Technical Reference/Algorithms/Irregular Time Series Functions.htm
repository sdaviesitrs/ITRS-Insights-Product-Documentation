Irregular Time Series Functions

Time Series is categorised as regular, when the data points come on a regular time based frequency (the spacing of "observations" is constant) or irregular when the data point frequency is not regular (the spacing of "observations" is uneven).

For example share trading data may be collected every hour; automatically, without fail and so would be categorised as regular time series data. However medical data such as temperature and blood pressure may be collected opportunistically when the patient, equipment and medical specialist are available, and so would likely be irregular.

Irregular data therefore is more complex to do statistical calculations on, requiring different algorithms. In Insights these all start with 'i' to signify irregular.

iEMA

Iterated (Recursive) EMA Operator

Format: iEMA(period = $$x$$ , n = $$n$$) (timestamp, value)

Parameters:

Name Type Comments
period double Range of the operator in units of time
n int Level of recursion
timestamp datetime Timestamp associated with the point being processed.
value double value of the time series at the given timestamp

iMicroTickFrequency

Average tick arrival over the given period

Format: iMicroTickFrequency(period)(timestamp, value)

Microscopic operator (that is, its value is a function of Î´t) that calculates the average of ticks arrived for the given period of time

Parameters

Name Type Comments
period double Range of the operator in units of time
timestamp datetime Timestamp associated with the point being processed.
value double value of the time series at the given timestamp

iMicroVolatility

Format: iMicroVolatility(period)(timestamp,value)

Microscopic Volatility, computed as the norm of the microscopic derivative of the signal
All microscopic operators are sensitive to the tick frequency allowing to extract information at the highest possible frequency.

Parameters

Name Type Comments
period double Range of the operator in units of time
timestamp datetime Timestamp associated with the point being processed.
value double value of the time series at the given timestamp

iMovDerivative

Moving Derivative for irregular time series

Format: iMovDerivative(period, exponent)

Parameters

Name Type Comments
period double Range of the operator in units of time
exponent double Use 1.0 for the usual derivative, 0.0 for a differential operator and 0.5 for a stochastic diffusion process
timestamp datetime Timestamp associated with the point being processed
value double value of the time series at the given timestamp

iMovAverage

Moving Average for irregular time series

Format: iMovAverage(period)(timestamp, value)

A moving average, is essentially a smoothing operator to find averages over a moving window. iMovingAverage uses interpolation to take into account irregular data. For example if the period was 1 minute and data arrived at the 10 and 50 second mark, the algo will take into consideration the relative times as well as the values.

Parameters:

Name Type Comments
period double Range of the operator in units of time
timestamp datetime Timestamp associated with the point being processed.
value double value of the time series at the given timestamp

Examples:

FROM /streams/demo/infrastructure/cpu
group by host, process
select host, process, avg(user) as au, iMovAverage(period = 2.0)(sampleTime, user) as ma

iMovSkewness

Returns the amount of skewness for an irregular time series. Skewness is a measure of the degree of asymmetry of a distribution. If the left tail (tail at small end of the distribution) is more pronounced than the right tail (tail at the large end of the distribution), the function is said to have negative skewness. If the reverse is true, it has positive skewness. If the two are equal, it has zero skewness.

Format: iMovSkewness(period, standardizationPeriod)(timestamp, value)

Parameters:

Name Type Comments
period double Range of the operator in units of time
standardizationPeriod double Range of the z-standardised operator
timestamp datetime Timestamp associated with the point being processed.
value double value of the time series at the given timestamp
